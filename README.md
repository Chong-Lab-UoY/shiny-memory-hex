# Microbes Match Game

A [Shiny](https://github.com/rstudio/shiny) app to find all matching microbes as soon as possible!

This application is based on the submission to the RStudio Shiny contest: https://community.rstudio.com/t/shiny-contest-submission-hex-memory-game/25336

To run the application locally (faster than online version below) use:

```r
shiny::runGitHub(repo = "Chong-Lab-UoY/shiny-memory-hex")
```
